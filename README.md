## Данный реопзиторий содержит проект, выполненный на базе SQL.

Заказчик: агентство недвиджимости.

Задача проекта: проанализировать объявления о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области, чтобы найти самые перспективные сегменты недвижимости.

Данные для анализа: архивная информация сервиса Яндекс Недвижимость: объявления о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области за несколько лет.

### Технологический стек:
- PostgreSQL
- DBeaver
