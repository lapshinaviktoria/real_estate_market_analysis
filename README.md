## Данный реопзиторий содержит проект, выполненный на базе SQL.

Заказчик: агентство недвиджимости.

Задачи проекта: 
1. Проанализировать объявления о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области, чтобы найти самые перспективные сегменты недвижимости.
2. Доработать существующий дашборд: добавить новые визуализации и фильтры 

Данные для анализа: архивная информация сервиса Яндекс Недвижимость: объявления о продаже жилой недвижимости в Санкт-Петербурге и Ленинградской области за несколько лет.

### Технологический стек:
- PostgreSQL
- DBeaver
- Yandex.Datalens
